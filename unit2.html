Part A: Short Answer Questions

1. Write a while control structure in a R programming with an example?

The while loop in R executes a block of code as long as a condition is TRUE.

  • Syntax:
    while (condition) {
  # Code block
}
  • Example:
    i <- 1
while (i <= 5) {
  print(i)
  i <- i + 1
}
# Outputs: [1] 1 [1] 2 [1] 3 [1] 4 [1] 5

    The loop prints numbers from 1 to 5, incrementing i until the condition i <= 5 is FALSE.

2. Write about next and break in R programming?

  • next: Skips the current iteration of a loop and moves to the next iteration.
    for (i in 1:5) {
  if (i == 3) next
  print(i)
}
# Outputs: [1] 1 [1] 2 [1] 4 [1] 5

    Here, next skips printing 3.

  • break: Exits the loop entirely when encountered.
    for (i in 1:5) {
  if (i == 3) break
  print(i)
}
# Outputs: [1] 1 [1] 2

    The loop stops when i equals 3.

3. Write short notes on built-in functions in R programming?

R provides numerous built-in functions for common tasks:

  • Mathematical: sum(), mean(), sqrt(), abs().
  • Statistical: sd(), var(), median().
  • Data Manipulation: length(), dim(), str().
  • Type Checking: is.numeric(), is.character().

Example:

x <- c(1, 2, 3, 4)
print(mean(x))  # Outputs 2.5
print(sqrt(16)) # Outputs 4

These functions simplify operations without requiring external packages.

4. How to declare the date and time functions in R programming?

R provides functions like Sys.Date(), Sys.time(), and packages like lubridate for handling dates and times.

  • Basic Functions:
    • Sys.Date(): Returns the current date.
    • Sys.time(): Returns the current date and time.
    • as.Date(): Converts strings to date objects.
    • format(): Formats date/time output.
  • Example:
    current_date <- Sys.Date()
current_time <- Sys.time()
custom_date <- as.Date("2025-06-26")
print(current_date)  # Outputs e.g., "2025-06-26"
print(format(current_time, "%Y-%m-%d %H:%M:%S"))  # Outputs e.g., "2025-06-26 18:19:00"

5. What are the variable scoping rules in R programming?

R follows lexical scoping:

  • Variables defined inside a function are local to that function.
  • Variables not found in the local environment are searched in the parent environment (e.g., global environment).
  • Use <<- for global assignment within functions.
  • Example:
    x <- 10  # Global variable
my_func <- function() {
  x <- 5  # Local variable
  print(x)
}
my_func()  # Outputs 5
print(x)   # Outputs 10 (global x unchanged)

    The <<- operator modifies global variables:

    my_func2 <- function() {
  x <<- 20
}
my_func2()
print(x)  # Outputs 20

6. Write about character strings in R programming?

Character strings in R are sequences of characters stored as the character data type.

  • Creation: Use quotes ("" or '').
  • Functions:
    • paste(), paste0(): Concatenate strings.
    • nchar(): Count characters.
    • toupper(), tolower(): Change case.
    • substr(): Extract substrings.
  • Example:
    str <- "Hello, R!"
print(nchar(str))      # Outputs 9
print(toupper(str))    # Outputs "HELLO, R!"
print(paste(str, "World"))  # Outputs "Hello, R! World"
print(substr(str, 1, 5))    # Outputs "Hello"

7. How to find the length of a vector with an example?

The length() function returns the number of elements in a vector.

  • Syntax: length(vector)
  • Example:
    vec <- c(10, 20, 30, 40)
print(length(vec))  # Outputs 4

For multi-dimensional objects like matrices, use dim() instead.

8. How to extracting elements of a vector using subscripts?

Elements in a vector can be extracted using square brackets [] with:

  • Numeric indices: Access specific positions.
  • Logical vectors: Select elements where TRUE.
  • Negative indices: Exclude elements.
  • Example:
    vec <- c(10, 20, 30, 40, 50)
print(vec[2])          # Outputs 20
print(vec[2:4])        # Outputs 20 30 40
print(vec[c(TRUE, FALSE, TRUE, FALSE, TRUE)])  # Outputs 10 30 50
print(vec[-1])         # Outputs 20 30 40 50 (excludes first)

9. How to deleting elements of matrices and arrays?

To delete elements, use subsetting to exclude specific indices:

  • Matrices: Use [rows, cols] to select rows/columns to keep.
  • Arrays: Use similar indexing for higher dimensions.
  • Example:
    # Matrix
mat <- matrix(1:9, nrow=3)
print(mat)
# Delete second row
mat_new <- mat[-2, ]
print(mat_new)  # Outputs matrix without row 2

# Array
arr <- array(1:12, dim=c(2, 3, 2))
arr_new <- arr[, -2, ]  # Delete second column
print(arr_new)

10. Write about scalars in R programming?

Scalars in R are single-element vectors:

  • R does not have a distinct scalar type; a scalar is a vector of length 1.
  • Used for simple values like numbers or strings.
  • Example:
    scalar <- 42
print(length(scalar))  # Outputs 1
print(scalar + 10)     # Outputs 52
str_scalar <- "Hello"
print(length(str_scalar))  # Outputs 1

Operations on scalars work like vector operations.

Part B: Long Answer Questions

11. a) Explain about control structures with an example in R programming.

Control structures in R manage the flow of execution. They include:

  • if-else: Conditional execution.
    x <- 10
if (x > 5) {
  print("x is greater than 5")
} else {
  print("x is 5 or less")
}
# Outputs: "x is greater than 5"
  • for: Iterates over a sequence.
    for (i in 1:3) {
  print(i^2)
}
# Outputs: [1] 1 [1] 4 [1] 9
  • while: Executes while a condition is true.
    i <- 1
while (i <= 3) {
  print(i)
  i <- i + 1
}
# Outputs: [1] 1 [1] 2 [1] 3
  • repeat: Loops indefinitely until break.
    i <- 1
repeat {
  print(i)
  i <- i + 1
  if (i > 3) break
}
# Outputs: [1] 1 [1] 2 [1] 3
  • next and break: Control loop iterations (see Q2).

11. b) Explain about arithmetic and logical operations on vectors.

  • Arithmetic Operations: R performs element-wise arithmetic on vectors.
    • +, -, *, /, ^: Operate on corresponding elements.
    • If vectors differ in length, the shorter one is recycled.
    • Example:
      a <- c(1, 2, 3)
b <- c(4, 5, 6)
print(a + b)  # Outputs 5 7 9
print(a * 2)  # Outputs 2 4 6
print(a ^ 2)  # Outputs 1 4 9
  • Logical Operations: Element-wise logical operations return TRUE/FALSE.
    • &, |, !: AND, OR, NOT.
    • Example:
      x <- c(TRUE, FALSE, TRUE)
y <- c(FALSE, TRUE, TRUE)
print(x & y)  # Outputs FALSE FALSE TRUE
print(x | y)  # Outputs TRUE TRUE TRUE
print(!x)     # Outputs FALSE TRUE FALSE

12. a) Explain about types of vectors with examples in R programming.

Vectors are the basic data structure in R, storing elements of the same type. Types include:

  • Numeric: Stores numbers (integers or doubles).
    num_vec <- c(1.5, 2.7, 3.2)
print(num_vec)  # Outputs 1.5 2.7 3.2
  • Integer: Whole numbers, specified with L.
    int_vec <- c(1L, 2L, 3L)
print(int_vec)  # Outputs 1 2 3
  • Character: Stores strings.
    char_vec <- c("apple", "banana")
print(char_vec)  # Outputs "apple" "banana"
  • Logical: Stores TRUE/FALSE.
    log_vec <- c(TRUE, FALSE, TRUE)
print(log_vec)  # Outputs TRUE FALSE TRUE
  • Complex: Stores complex numbers.
    comp_vec <- c(1+2i, 3+4i)
print(comp_vec)  # Outputs 1+2i 3+4i

12. b) Write an R program to create a vector and to access elements in a vector.

# Creating vectors
num_vec <- c(10, 20, 30, 40, 50)
char_vec <- c("A", "B", "C")

# Accessing elements
print("Numeric Vector:")
print(num_vec[2])       # Outputs 20
print(num_vec[1:3])     # Outputs 10 20 30
print(num_vec[c(1, 5)]) # Outputs 10 50

print("Character Vector:")
print(char_vec[2])      # Outputs "B"
print(char_vec[-1])     # Outputs "B" "C" (excludes first)

# Logical indexing
print(num_vec[num_vec > 25])  # Outputs 30 40 50

13. a) Explain about vector operations in R programming with examples.

Vector operations in R are element-wise and include:

  • Arithmetic: +, -, *, /, ^.
    a <- c(1, 2, 3)
b <- c(4, 5, 6)
print(a + b)  # Outputs 5 7 9
print(a * b)  # Outputs 4 10 18
  • Logical: &, |, !.
    x <- c(TRUE, FALSE)
y <- c(FALSE, TRUE)
print(x & y)  # Outputs FALSE FALSE
  • Comparison: ==, !=, >, <.
    print(a > b)  # Outputs FALSE FALSE FALSE
  • Recycling: If vectors differ in length, the shorter is recycled.
    c <- c(2)
print(a + c)  # Outputs 3 4 5 (2 added to each element)

13. b) Write a program to create a matrix using cbind() and rbind() functions.

# Using cbind() to create a matrix by binding columns
col1 <- c(1, 2, 3)
col2 <- c(4, 5, 6)
mat_cbind <- cbind(col1, col2)
print("Matrix using cbind():")
print(mat_cbind)
# Outputs:
#      col1 col2
# [1,]    1    4
# [2,]    2    5
# [3,]    3    6

# Using rbind() to create a matrix by binding rows
row1 <- c(7, 8)
row2 <- c(9, 10)
mat_rbind <- rbind(row1, row2)
print("Matrix using rbind():")
print(mat_rbind)
# Outputs:
#      [,1] [,2]
# row1    7    8
# row2    9   10

14. a) Explain about functions in R programming.

Functions in R are objects that perform specific tasks.

  • Definition: Use function() to create.
    square <- function(x) {
  return(x^2)
}
  • Calling: Use the function name with arguments.
    print(square(5))  # Outputs 25
  • Default Arguments: Specify default values.
    add <- function(a, b=10) {
  return(a + b)
}
print(add(5))     # Outputs 15 (uses default b=10)
print(add(5, 20)) # Outputs 25
  • Anonymous Functions: Used without naming.
    (function(x) x*2)(3)  # Outputs 6

14. b) Explain about vectors and subscripts and generating sequences in R programming.

  • Vectors: One-dimensional arrays of the same type.
    vec <- c(1, 2, 3)
  • Subscripts: Access elements using [].
    • Numeric: vec[2] (outputs 2).
    • Logical: vec[vec > 1] (outputs 2 3).
    • Negative: vec[-1] (excludes first element).
  • Generating Sequences:
    • :: Creates sequences with step 1.
      seq1 <- 1:5  # Outputs 1 2 3 4 5
    • seq(): Custom sequences.
      seq2 <- seq(from=1, to=10, by=2)  # Outputs 1 3 5 7 9
    • rep(): Repeats elements.
      rep_vec <- rep(1, times=3)  # Outputs 1 1 1

15. a) Explain about multiple and single element vector creation in R programming.

  • Single Element Vector (Scalar):
    single <- 42
print(length(single))  # Outputs 1
print(single)         # Outputs 42
  • Multiple Element Vector:
    • Use c() to combine elements.
    • Use sequences or repetition.
    • Example:
      multi <- c(10, 20, 30)
print(multi)  # Outputs 10 20 30
seq_vec <- 1:5
print(seq_vec)  # Outputs 1 2 3 4 5
rep_vec <- rep("A", 3)
print(rep_vec)  # Outputs "A" "A" "A"

15. b) Explain about adding and deleting vector elements. With examples.

  • Adding Elements:
    • Use c() to append elements.
    • Use indexing to insert at specific positions.
    • Example:
      vec <- c(1, 2, 3)
vec <- c(vec, 4)      # Append 4
print(vec)            # Outputs 1 2 3 4
vec[5] <- 5           # Add at index 5
print(vec)            # Outputs 1 2 3 4 5
  • Deleting Elements:
    • Use negative indexing to exclude elements.
    • Example:
      vec <- c(10, 20, 30, 40)
vec <- vec[-2]        # Delete second element
print(vec)            # Outputs 10 30 40
vec <- vec[-c(1, 3)]  # Delete multiple elements
print(vec)            # Outputs 30

Introduction to R Programming

R is a powerful open-source programming language for statistical computing and data visualization. It's widely used for data analysis, machine learning, and academic research. Below are key concepts to get started with R:

  • Variables: Store data using <- or =.
  • Data Structures: Vectors, matrices, data frames, and lists.
  • Functions: Built-in or user-defined for reusable code.
  • Packages: Extend R's functionality (e.g., ggplot2, dplyr).

Example: Basic R Operations

# Create a vector
numbers <- c(1, 2, 3, 4, 5)

# Calculate mean and plot
mean_value <- mean(numbers)
plot(numbers, type = "l", col = "blue", main = "Simple Line Plot", xlab = "Index", ylab = "Value")
# Output: Mean = 3

Part A: Short Answer Questions

1. Define the list. Give an examples.

A list in R is a versatile data structure that can store elements of different types (e.g., numeric, character, logical, or even other lists). Unlike vectors, lists can hold heterogeneous data.

Example:

my_list <- list(name = "Alice", age = 25, scores = c(90, 85, 88))
print(my_list)
# Output:
# $name
# [1] "Alice"
# $age
# [1] 25
# $scores
# [1] 90 85 88

2. How to create the list in R programming?

Lists are created using the list() function, where elements can be named or unnamed.

  • Syntax: list(name1 = value1, name2 = value2, ...)
  • Example:
    # Named list
student <- list(name = "Bob", age = 30, active = TRUE)
print(student)
# Unnamed list
simple_list <- list(1, "hello", TRUE)
print(simple_list)
# Output:
# $name
# [1] "Bob"
# $age
# [1] 30
# $active
# [1] TRUE
#
# [[1]]
# [1] 1
# [[2]]
# [1] "hello"
# [[3]]
# [1] TRUE

3. Write about list indexing adding and deleting list elements in R programming?

  • Indexing:
    • Use [[]] to access a single element (returns the element itself).
    • Use [] to return a sublist.
    • Use $ for named elements.
    • Example:
      my_list <- list(name = "Alice", age = 25)
print(my_list[["name"]])  # Outputs "Alice"
print(my_list$age)        # Outputs 25
print(my_list[1])         # Outputs sublist: $name [1] "Alice"
  • Adding Elements:
    • Use [[index]] or $name to add new elements.
    • Example:
      my_list$new_element <- "new"
print(my_list)  # Adds $new_element with "new"
  • Deleting Elements:
    • Set elements to NULL or use negative indexing.
    • Example:
      my_list$age <- NULL
print(my_list)  # Age is removed

4. How to find the size of a list with an example?

The length() function returns the number of top-level elements in a list.

  • Syntax: length(list)
  • Example:
    my_list <- list(a = 1, b = c(2, 3), c = "hello")
print(length(my_list))  # Outputs 3

5. How to access the list elements from the list give an example?

List elements can be accessed using:

  • Double brackets [[]]: Returns the element’s content.
  • Single brackets []: Returns a sublist.
  • Dollar sign $: Access named elements.
  • Example:
    my_list <- list(name = "Alice", scores = c(90, 85), active = TRUE)
print(my_list[[1]])      # Outputs "Alice"
print(my_list$scores)    # Outputs 90 85
print(my_list[1:2])      # Outputs sublist with name and scores

6. How to create data frames in a R programming?

A data frame is a table-like structure where columns can have different data types.

  • Syntax: data.frame(col1 = values, col2 = values, ...)
  • Example:
    df <- data.frame(
  name = c("Alice", "Bob", "Charlie"),
  age = c(25, 30, 22),
  score = c(90, 85, 88)
)
print(df)
# Output:
#      name age score
# 1   Alice  25    90
# 2     Bob  30    85
# 3 Charlie  22    88

7. How to create matrix in R programming?

A matrix is a 2D array with elements of the same type, created using matrix().

  • Syntax: matrix(data, nrow, ncol, byrow = FALSE)
  • Example:
    mat <- matrix(1:6, nrow = 2, ncol = 3)
print(mat)
# Output:
#      [,1] [,2] [,3]
# [1,]    1    3    5
# [2,]    2    4    6

8. Write about matrix operations in R programming?

Matrix operations include:

  • Arithmetic: Element-wise addition, subtraction, multiplication, etc.
    mat1 <- matrix(1:4, nrow = 2)
mat2 <- matrix(5:8, nrow = 2)
print(mat1 + mat2)  # Element-wise addition
  • Matrix Multiplication: Use %*% for matrix multiplication.
    print(mat1 %*% t(mat2))  # Matrix multiplication with transpose
  • Transpose: t(matrix) transposes rows and columns.
    print(t(mat1))
  • Inverse: solve(matrix) for square matrices.
    mat <- matrix(c(1, 2, 3, 4), nrow = 2)
print(solve(mat))  # Inverse

9. How to access the elements from the data frame in R programming?

Data frame elements can be accessed using:

  • Column names with $: df$column.
  • Brackets [row, col]: Specify row and column indices or names.
  • Double brackets [[]]: Access columns.
  • Example:
    df <- data.frame(name = c("Alice", "Bob"), age = c(25, 30))
print(df$name)          # Outputs "Alice" "Bob"
print(df[1, 2])         # Outputs 25
print(df[, "age"])      # Outputs 25 30
print(df[[2]])          # Outputs 25 30

10. How to find the length of the data frame in R programming?

  • Use nrow() for the number of rows.
  • Use ncol() for the number of columns.
  • Use dim() for both dimensions.
  • Example:
    df <- data.frame(name = c("Alice", "Bob"), age = c(25, 30))
print(nrow(df))  # Outputs 2
print(ncol(df))  # Outputs 2
print(dim(df))   # Outputs 2 2

Part B: Long Answer Questions

11. a) Explain about list operations with examples.

List operations in R include creating, accessing, modifying, and deleting elements:

  • Creating: Use list().
    my_list <- list(a = 1, b = "hello", c = c(2, 3))
  • Accessing: Use [[]], [], or $.
    print(my_list$a)         # Outputs "hello"
print(my_list[[3]])      # Outputs 2 3
print(my_list[1:2])      # Outputs sublist with a and b
  • Modifying: Assign new values to existing or new indices.
    my_list$a <- "new_value"
print(my_list$a)         # Outputs "new_value"
  • Adding: Use $ or [[index]].
    my_list$new <- TRUE
print(my_list$new)       # Outputs TRUE
  • Deleting: Set elements to NULL.
    my_list$b <- NULL
print(my_list)           # b is removed

12. a) Explain about data frame operations in R programming.

Data frame operations include:

  • Creating: Use data.frame().
    df <- data.frame(x = 1:3, y = c("a", "b", "c"))
  • Accessing: Use $, [[]], or [row, col].
    print(df$y)       # Outputs "a" "b" "c"
print(df[2, 1])   # Outputs 2
  • Adding Columns: Assign new columns with $.
    df$z <- c(4, 5, 6)
  • Adding Rows: Use rbind().
    new_row <- data.frame(x = 4, y = "d", z = 7)
df <- rbind(df, new_row)
  • Deleting: Use NULL for columns or subsetting for rows.
    df$z <- NULL           # Remove column z
df <- df[-1, ]         # Remove first row
  • Filtering: Subset using logical conditions.
    print(df[df$x > 2, ])  # Rows where x > 2

12. b) Write an R program to access a data frame like a list.

# Creating a data frame
df <- data.frame(
  name = c("Alice", "Bob", "Charlie"),
  age = c(25, 30, 22)
)

# Accessing like a list
print("Accessing name column using $:")
print(df$name)          # Outputs "Alice" "Bob" "Charlie"

print("Accessing age column using [[]]:")
print(df[["age"]])      # Outputs 25 30 22

print("Accessing first two columns as a sublist:")
print(df[1:2])          # Outputs name and age columns

print("Accessing first element of name column:")
print(df$name[1])       # Outputs "Alice"

13. a) Explain about list with data frame give examples.

A list can contain a data frame as one of its elements, allowing complex data organization.

Example:

df <- data.frame(name = c("Alice", "Bob"), age = c(25, 30))
my_list <- list(student_info = df, course = "Statistics")
print(my_list)
# Output:
# $student_info
#    name age
# 1 Alice  25
# 2   Bob  30
# $course
# [1] "Statistics"

# Accessing
print(my_list$student_info$name)  # Outputs "Alice" "Bob"
print(my_list[[1]][, "age"])      # Outputs 25 30

13. b) Explain about creating and adding elements to the list give an example.

  • Creating: Use list() to create a list with named or unnamed elements.
    my_list <- list(a = 1, b = "hello")
  • Adding Elements:
    • Use $ to add named elements.
    • Use [[index]] for specific positions.
    • Use c() to append lists.
    • Example:
      my_list <- list(a = 1, b = "hello")
my_list$c <- TRUE          # Add named element
my_list[[4]] <- c(2, 3)    # Add at index 4
my_list <- c(my_list, list(d = "new"))  # Append another list
print(my_list)
# Output:
# $a
# [1] 1
# $b
# [1] "hello"
# $c
# [1] TRUE
# [[4]]
# [1] 2 3
# $d
# [1] "new"

14. b) Explain about accessing and deleting elements from the data frame give an example.

    • Use $ for columns.

15. a) How to convert list to matrix in R programming.

To convert a list to a matrix, ensure the list contains compatible data (e.g., numeric vectors of equal length) and use matrix() or do.call().

Example:

my_list <- list(c(1, 2, 3), c(4, 5, 6))
mat <- do.call(cbind, my_list)  # Combine list elements as columns
print(mat)
# Output:
#      [,1] [,2]
# [1,]    1    4
# [2,]    2    5
# [3,]    3    6

Alternatively, use matrix() with unlist():

mat <- matrix(unlist(my_list), nrow = 3, byrow = FALSE)

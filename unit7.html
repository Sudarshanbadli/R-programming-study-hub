Part A: Short Answer Questions

1. What is a T-test in R, and how is it performed?

A T-test compares the means of one or two groups to determine if they are significantly different. In R, use t.test().

Example (One-sample T-test):

data <- c(10, 12, 14, 11, 13)
t.test(data, mu = 12)  # Test if mean equals 12
# Output: p-value indicates significance

2. What is an ANOVA test in R, and when is it used?

ANOVA (Analysis of Variance) tests if there are significant differences between the means of three or more groups. Use aov() or anova().

Example:

data <- data.frame(value = c(10, 12, 14, 20, 22, 24, 30, 32, 34),
                           group = rep(c("A", "B", "C"), each = 3))
result <- aov(value ~ group, data = data)
summary(result)

3. What is covariance in R, and how is it calculated?

Covariance measures the linear relationship between two variables. Use cov() in R.

Example:

x <- c(1, 2, 3, 4, 5)
y <- c(2, 4, 6, 8, 10)
print(cov(x, y))  # Outputs: 5 (positive covariance)

4. What is correlation in R, and how is it computed?

Correlation measures the strength and direction of the linear relationship between two variables, ranging from -1 to 1. Use cor() in R.

Example:

x <- c(1, 2, 3, 4, 5)
y <- c(2, 4, 6, 8, 10)
print(cor(x, y))  # Outputs: 1 (perfect positive correlation)

5. How is error handling implemented in R?

Error handling in R is managed using try(), tryCatch(), or stop() to catch and handle errors gracefully.

Example:

result <- try(log(-1))  # Invalid input
if (inherits(result, "try-error")) {
  print("Error: Logarithm of negative number")
}
# Output: Error: Logarithm of negative number

6. What is a reference class in R?

Reference classes (RC) in R provide object-oriented programming with mutable objects and methods, created using setRefClass().

Example:

Person <- setRefClass("Person", fields = list(name = "character"),
                              methods = list(show = function() cat("Name:", name, "\n")))
p <- Person$new(name = "Alice")
p$show()  # Outputs: Name: Alice

7. What is descriptive analysis in R?

Descriptive analysis summarizes data using statistics like mean, median, mode, min, max, and percentiles, often visualized with plots.

Example:

data <- c(1, 2, 2, 3, 4)
summary(data)  # Outputs min, max, quartiles, mean

8. What is hypothesis testing in R?

Hypothesis testing evaluates whether sample data supports a null hypothesis (H₀) against an alternative (H₁) using tests like T-test or ANOVA.

Example (Two-sample T-test):

group1 <- c(10, 12, 14)
group2 <- c(15, 17, 19)
t.test(group1, group2)  # Compare means

9. What is linear regression in R, and how is it performed?

Linear regression models the relationship between a dependent variable and one or more independent variables. Use lm() in R.

Example:

x <- 1:5
y <- c(2, 4, 5, 4, 5)
model <- lm(y ~ x)
summary(model)

10. How do you interpret the output of a linear regression in R?

The output of lm() includes:

  • Coefficients: Slope and intercept.
  • R-squared: Proportion of variance explained.
  • p-value: Significance of predictors.

Example:

model <- lm(mpg ~ wt, data = mtcars)
summary(model)  # Outputs coefficients, R-squared, p-values

Part B: Long Answer Questions

11. a) Explain the T-test and its applications in R.

A T-test compares means to assess significant differences:

  • One-sample T-test: Tests if a sample mean differs from a known value (t.test(x, mu = value)).
  • Two-sample T-test: Compares means of two groups (t.test(x, y)).
  • Paired T-test: Tests differences in paired data (t.test(x, y, paired = TRUE)).
  • Applications: Comparing group means in experiments (e.g., drug efficacy).

Example:

group1 <- c(10, 12, 14, 11, 13)
group2 <- c(15, 17, 19, 16, 18)
t.test(group1, group2)  # Two-sample T-test

11. b) Write an R program to perform a two-sample T-test and interpret results.

# Sample data
group1 <- c(10, 12, 14, 11, 13)
group2 <- c(15, 17, 19, 16, 18)

# Perform two-sample T-test
result <- t.test(group1, group2, var.equal = TRUE)

# Display results
print(result)

# Interpret
cat("p-value:", result$p.value, "\n")
if (result$p.value < 0.05) {
  cat("Reject null hypothesis: Means are significantly different\n")
} else {
  cat("Fail to reject null hypothesis: No significant difference\n")
}
# Output example:
# p-value: 0.0012
# Reject null hypothesis: Means are significantly different

12. a) Explain ANOVA and its use in R.

ANOVA tests differences between means of three or more groups:

  • One-way ANOVA: Compares means across one factor (aov(y ~ factor)).
  • Two-way ANOVA: Includes two factors.
  • Use: Compare multiple groups (e.g., treatment effects across groups).

Example:

data <- data.frame(value = c(10, 12, 14, 20, 22, 24, 30, 32, 34),
                           group = rep(c("A", "B", "C"), each = 3))
result <- aov(value ~ group, data = data)
summary(result)

12. b) Write an R program to perform a one-way ANOVA and visualize results.

# Sample data
data <- data.frame(value = c(10, 12, 14, 20, 22, 24, 30, 32, 34),
                   group = rep(c("A", "B", "C"), each = 3))

# Perform one-way ANOVA
result <- aov(value ~ group, data = data)
print(summary(result))

# Visualize with boxplot
boxplot(value ~ group, data = data, main = "Boxplot by Group", 
        xlab = "Group", ylab = "Value", col = c("lightblue", "lightgreen", "lightpink"))
# Output: p-value indicates if group means differ significantly

13. a) Explain covariance and correlation in R.

  • Covariance: Measures the linear relationship between two variables (cov()). Positive values indicate variables move together; negative values indicate opposite movement.
  • Correlation: Standardizes covariance to range from -1 to 1 (cor()). Methods include Pearson (linear) and Spearman (rank-based).

Example:

x <- c(1, 2, 3, 4, 5)
y <- c(2, 4, 6, 8, 10)
print(cov(x, y))   # Outputs: 5
print(cor(x, y))   # Outputs: 1

13. b) Write an R program to compute and visualize covariance and correlation.

# Sample data
x <- c(1, 2, 3, 4, 5)
y <- c(2, 4, 6, 8, 10)

# Compute covariance and correlation
cov_xy <- cov(x, y)
cor_xy <- cor(x, y, method = "pearson")

# Display results
cat("Covariance:", cov_xy, "\n")
cat("Correlation:", cor_xy, "\n")

# Visualize with scatterplot
plot(x, y, main = "Scatterplot of X vs Y", xlab = "X", ylab = "Y", 
     pch = 19, col = "blue")
abline(lm(y ~ x), col = "red")  # Add regression line
# Output:
# Covariance: 5
# Correlation: 1

14. a) Explain error handling in R and its importance.

Error handling in R ensures robust programs by managing exceptions:

  • Functions: try(), tryCatch(), stop(), warning().
  • Use: Prevents crashes, provides user-friendly messages, and allows recovery.

Example:

tryCatch(log(-1), error = function(e) cat("Error:", e$message, "\n"))
# Outputs: Error: non-positive x

14. b) Write an R program to implement error handling for invalid inputs.

# Function with error handling
safe_log <- function(x) {
  tryCatch({
    if (x <= 0) stop("Input must be positive")
    return(log(x))
  }, error = function(e) {
    cat("Error:", e$message, "\n")
    return(NULL)
  })
}

# Test cases
print(safe_log(10))    # Outputs: 2.302585
print(safe_log(-1))    # Outputs: Error: Input must be positive
print(safe_log("abc")) # Outputs: Error: non-numeric argument

15. a) Explain descriptive analysis and hypothesis testing in R.

  • Descriptive Analysis: Summarizes data using statistics (mean, median, mode, percentiles) and visualizations (boxplots, histograms).
  • Hypothesis Testing: Tests hypotheses about population parameters using tests like T-test, ANOVA, or chi-square tests.

Example:

data <- c(10, 12, 14, 11, 13)
summary(data)  # Descriptive stats
t.test(data, mu = 12)  # Hypothesis test

15. b) Write an R program to perform descriptive analysis and hypothesis testing.

# Sample data
data <- c(10, 12, 14, 11, 13, 15, 16, 14, 12)

# Descriptive analysis
cat("Mean:", mean(data), "\n")
cat("Median:", median(data), "\n")
cat("Standard Deviation:", sd(data), "\n")
cat("Percentiles:\n")
print(quantile(data, probs = c(0.25, 0.5, 0.75)))

# Hypothesis testing (one-sample T-test)
result <- t.test(data, mu = 12)
print(result)

# Visualize with histogram
hist(data, main = "Histogram of Data", xlab = "Values", col = "lightblue")
# Output example:
# Mean: 13
# Median: 13
# Standard Deviation: 1.870829
# Percentiles:
# 25% 50% 75%
# 12  13  14.5
